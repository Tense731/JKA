Here's an example of how to create a dropdown menu that displays continents when clicked using HTML, CSS, and JavaScript:

*HTML:*

```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Continents Dropdown</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <div class="dropdown">
        <button class="dropbtn">Continents</button>
        <div class="dropdown-content" id="dropdown-content">
            <!-- continents will be listed here -->
        </div>
    </div>

    <script src="script.js"></script>
</body>
</html>
```

*CSS (style.css):*

```
.dropdown {
    position: relative;
    display: inline-block;
}

.dropdown-content {
    display: none;
    position: absolute;
    background-color: #f9f9f9;
    min-width: 160px;
    box-shadow: 0px 8px 16px 0px rgba(0,0,0,0.2);
    padding: 12px 16px;
    z-index: 1;
}

.dropdown-content a {
    color: black;
    padding: 12px 16px;
    text-decoration: none;
    display: block;
}

.dropdown-content a:hover {background-color: #f1f1f1}

.dropbtn {
    background-color: #4CAF50;
    color: white;
    padding: 16px;
    font-size: 16px;
    border: none;
    cursor: pointer;
}

.dropbtn:hover {
    background-color: #3e8e41;
}
```

*JavaScript (script.js):*

```
const continents = ["Africa", "Antarctica", "Asia", "Australia/Oceania", "Europe", "North America", "South America"];

document.querySelector(".dropbtn").addEventListener("click", () => {
    const dropdownContent = document.getElementById("dropdown-content");
    dropdownContent.innerHTML = "";
    continents.forEach(continent => {
        const a = document.createElement("a");
        a.textContent = continent;
        dropdownContent.appendChild(a);
    });
    dropdownContent.style.display = dropdownContent.style.display === "block" ? "none" : "block";
});
```

*Explanation:*

1. Create an HTML file with a `div` containing a `button` and an empty `div` for the dropdown content.
2. Add CSS styles to design the dropdown menu.
3. In JavaScript, define an array of continents.
4. Attach an event listener to the button that toggles the dropdown content.
5. When clicked, create and append `a` elements for each continent to the dropdown content.

*Example Use Case:*

You can use this code to create a dropdown menu for selecting continents on a geography-related website or application.
